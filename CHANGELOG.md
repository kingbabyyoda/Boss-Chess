# Changelog

## 1.0.0

Boss Chess 1.0.0 is the first stable release.

Highlights:
- Modular multi-file project structure
- Terminal and GUI launchers
- Trainer, meme, and cheat modes
- Stronger search with iterative deepening and endgame probing
- Polished GUI with animations, promotion chooser, and live status panels
- Persisted preferences, saved geometry, and crash reports
- Release workflow for Windows, macOS, and Linux builds
